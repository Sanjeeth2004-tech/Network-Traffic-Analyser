# Network Traffic Analyzer (Java)

A Network Traffic Analyzer is a Java-based application designed to capture, monitor, and analyze network traffic in real time. The project helps in understanding packet-level communication, protocol behavior, and traffic patterns within a network.

This project is developed as part of the **Computer Networks domain**, focusing on the practical application of networking concepts using Java.
